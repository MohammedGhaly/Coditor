About

The inspiration for Coditor project arose from my own experiences as a budding programmer. I vividly recall countless instances when I wished I could collaborate with friends or mentors in real-time, working on code together, and receiving immediate feedback. This desire for seamless collaboration and the excitement of witnessing code come to life instantly motivated me to embark on this journey. I wanted to create a platform that fosters learning, creativity, and teamwork, offering a space where coding enthusiasts from across the globe could connect, code together, and bring their ideas to fruition in real-time. This project is a manifestation of my passion for coding and my desire to make collaborative coding accessible to all. This project was developed as apart of ALX Software Engineering program.
